> 우리 팀이 탁! 맞는 순간, 티키탁

<img width="1140" height="750" alt="메인 포스터" src="https://github.com/user-attachments/assets/9381dc40-0a57-4c10-8cb1-626e4d98ebf7" />
티키탁(Tiki-Tak) 은 대학생의 팀 프로젝트 과정의 순간들을 함께 기록하고 회고하는 팀 단위 아카이브 서비스입니다.

<br/>

## 📱 주요 기능

### 🙋 오늘의 질문

매일 주어지는 질문에 답하며 데일리 피드로 기록하고 공유합니다.

<img src="https://github.com/user-attachments/assets/f5c71352-700e-46a7-8a14-feca283fb058" alt="오늘의 질문" width="100%" />

### 📅 활동 페이지

흩어진 팀의 기록을 한 달 단위로 큐레이션해 다시 꺼내보는 회고를 만듭니다.

<img src="https://github.com/user-attachments/assets/b19b4f2a-69d7-4a56-a82d-7fffac100a71" alt="활동 페이지" width="100%" />

### 🗺️ 지도 / 피드

동일한 기록 데이터를 시간순(피드)과 장소순(지도) 두 가지 진입점으로 조회합니다.

<img src="https://github.com/user-attachments/assets/97b660b6-5eb8-4d19-a239-f20f47d8f58b" alt="지도 / 피드" width="100%" />

### ✍️ 글쓰기

사진·텍스트·위치·팀원 태그를 하나의 게시물로 묶어 흩어지지 않게 기록합니다.

<img src="https://github.com/user-attachments/assets/c0fe8de8-5b48-4eb5-b18f-289fb8e0a5b6" alt="글쓰기" width="100%" />

### 👬 팀 관리

팀명과 한 줄 소개만으로 간단히 팀을 만들고, QR·링크로 팀원을 초대합니다.

<img src="https://github.com/user-attachments/assets/a2480dc6-1e67-46f1-a756-d70bf048330e" alt="팀 관리" width="100%" />
